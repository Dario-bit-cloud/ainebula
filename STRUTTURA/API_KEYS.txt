6. DOVE SONO CONFIGURATE
--------------------------------------------------------------------------------

FILE PRINCIPALI:

1. src/config/api.js
   - API_CONFIG.apiKey (Electron Hub)
   - LLM7_CONFIG.apiKey (LLM7.io)
   - MODEL_MAPPING (mappatura modelli)

2. server.js
   - DATABASE_URL (da process.env o hardcoded)
   - JWT_SECRET (da process.env o default)
   - WEBAUTHN_RP_ID (da process.env o default)
   - WEBAUTHN_ORIGIN (da process.env o default)
   - WEBAUTHN_RP_NAME (da process.env o default)

3. .env (file non committato, creare manualmente)
   - DATABASE_URL=...
   - JWT_SECRET=...
   - WEBAUTHN_RP_ID=...
   - WEBAUTHN_ORIGIN=...
   - WEBAUTHN_RP_NAME=...

4. Vercel Dashboard (variabili d'ambiente produzione)
   - Configurare in Settings → Environment Variables

--------------------------------------------------------------------------------
7. SICUREZZA
--------------------------------------------------------------------------------

⚠️ IMPORTANTE:

1. NON COMMITTARE API KEYS NEL REPOSITORY
   - Usa variabili d'ambiente per produzione
   - File .env non deve essere committato (già in .gitignore)

2. ROTAZIONE PERIODICA
   - Cambia API keys periodicamente
   - Revoca chiavi compromesse immediatamente

3. PERMESSI MINIMI
   - Usa API keys con permessi limitati
   - Non usare chiavi admin per operazioni normali

4. MONITORAGGIO
   - Monitora uso API keys
   - Imposta alert per uso anomalo
   - Controlla rate limits

5. JWT_SECRET
   - Deve essere unico e sicuro
   - Mai condiviso o esposto
   - Cambia in produzione

--------------------------------------------------------------------------------
8. COME OTTENERE NUOVE API KEYS
--------------------------------------------------------------------------------

ELECTRON HUB:
1. Vai su https://playground.electronhub.ai/console
2. Accedi o registrati
3. Vai alla sezione API Keys
4. Crea nuova chiave
5. Copia e incolla in src/config/api.js

LLM7.IO:
1. Vai su https://llm7.io
2. Accedi o registrati
3. Vai alla sezione API Keys
4. Crea nuova chiave
5. Copia e incolla in src/config/api.js

OPENROUTER (se necessario):
1. Vai su https://openrouter.ai
2. Accedi con Google/Discord/Email
3. Vai su https://openrouter.ai/keys
4. Crea nuova chiave
5. Configura in src/config/api.js

NEON POSTGRESQL:
1. Vai su https://neon.tech
2. Accedi al tuo progetto
3. Vai su Connection Details
4. Copia connection string
5. Configura in DATABASE_URL
